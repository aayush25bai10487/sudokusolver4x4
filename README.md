# Sudoku Solver
A simple 4x4 Sudoku Solver made in python




## Installation

Install numpy library on the python interpreter:

```bash
  pip install numpy
  OR
  python -m pip install numpy
  OR
  python3 -m pip install numpy
```
To check if numpy library is installed:

```bash
  pip show numpy
  OR
  python pip show numpy
  OR
  python3 pip show numpy
```

## Features

- Backtracking to Solve simple and complex 4x4 Sudokus.
- Easy to use keyboard inputs for inputting problems and getting solution.

- Python Interpreter
- Python Library (Numpy)
